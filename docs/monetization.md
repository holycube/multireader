## 1. 决策摘要

| 项 | 结论 | 生效版本 |
|----|------|----------|
| 激励广告 | **取消**；不接入 AdMob / 任何广告 SDK | v1.0 |
| 40 块阅读墙 | **取消**；导入后全书块均可预处理与阅读 | v1.0 |
| 首发平台 | **Google Play 海外**（Android）；iOS 延后 | v1.0 |
| 变现模型 | **免费核心 + 一次性 Pro 买断** + 可选内容包 IAP | v1.1 起开卖 |
| Pro 功能顺序 | Anki 导出 → 匿名备份 → 大词典包 | v1.1 → v1.2 → v1.3 |

**一句话**：v1.0 做完整、无广告、无阅读硬墙的免费阅读器；Pro 卖「深度学习用户的扩展能力」，不卖阅读权限。

## 2. 与不背单词路线的对齐

| 维度 | 不背单词（参考） | 本产品 |
|------|------------------|--------|
| 免费层 | 核心背单词够用 | 导入、阅读、查词、高亮、内置词典、基础统计 |
| 付费层 | 词书包、高级能力 | Pro 永久：Anki 导出、备份、大词典包等 |
| 订阅 vs 买断 | 偏订阅 | **Pro 永久买断** + 词典等内容包可单独 IAP |
| 体验原则 | 主流程无插播广告 | 同上；v1.0 零广告 SDK |
| 更新节奏 | 免费功能迭代 + 付费增量 | v1.x 小步追加 Pro 能力，文档与商店说明同步 |

## 3. 版本与功能边界

### 3.1 v1.0 免费（Google Play 海外首发）

| 包含 | 不含（后续 Pro / 内容包） |
|------|---------------------------|
| EPUB/TXT 导入与全书阅读 | Anki `.apkg` 导出 |
| 生词高亮 + 查词 + 词形双 Tab | 匿名进度备份 |
| 词库向导 + 生词本列表 | 完整 ECDICT / 大词典包按需下载 |
| 阅读统计（时长、今日新词、7 日趋势） | 云同步、账号体系 |
| 内置 ~10k MVP 词典 + 别名表 | 运行时词干化（P2） |
| 个人中心、合规静态页 | — |

### 3.2 Pro 永久（v1.1+，一次性 IAP）

| 顺序 | 功能 | 目标版本 | 说明 |
|------|------|----------|------|
| 1 | **Anki 导出** | v1.1 | 生词本 → `.apkg`；个人中心勋章墙解锁 |
| 2 | **匿名备份** | v1.2 | 二维码 / 字符串；换机恢复词库与进度 |
| 3 | **大词典包** | v1.3 | 按需下载完整裁剪词典；可与 v1.0 词典拆包共用 CDN 管线 |

**定价**：首发前在 Google Play Console 定档（建议 Pro 永久 \$4.99–\$9.99 区间手测；内容包单独定价）。

**技术占位**（v1.1 前可只做接口，v1.0 不展示购买入口亦可）：

- `poc/lib/services/entitlement_service.dart` — 本地 Pro 状态（`SharedPreferences` + Play Billing 校验）
- `parse_quota` — **保留表结构**；v1.0 不再用于拦阅读；未来若需「超大书预处理配额」可复用，**不**作为 v1.1 Pro 主卖点

### 3.3 明确不卖

- 阅读章节/块解锁（已取消广告墙）
- 横幅 / 插屏 / 激励视频广告
- 强制账号与上传阅读内容
- 书源聚合与盗版分发

| 区域 | v1.0 处理 | 路径 |

## 5. Google Play 海外首发（v1.0）

| 项 | 说明 |
|----|------|
| 开发者账号 | 一次性 **\$25**；审核通常 1–3 天 |
| 包体 | 目标 **&lt; 35MB** arm64 download；Sprint 12 Release **剔除词典 assets**（~5.5 MB）；Noto 字体 ~22 MB 仍可能超标 → Play Console 实测 |
| 首次启动 | Release 需联网下载词典包（~5.5 MB）；Debug 无网络依赖 |
| 隐私政策 | 需**可公开 URL**（GitHub Pages / 静态站）；App 内 Markdown 不够 |
| 数据安全表单 | 声明本地存储、无广告 SDK、无账号 |
| 分级 | IARC 问卷；无 UGC、无暴力 → 低年龄友好 |
| 签名 | Release keystore 妥善备份；**丢失无法更新同一包名** |

**v1.0 不需要**：软著、ICP 备案、国内渠道资质。

## 6. 国内市场参考（暂不首发，提前了解）

> 仅作预算与排期参考；**v1.0 不做国内渠道时可并行准备，不阻塞 Play 海外**。

### 6.1 软件著作权（软著）

| 方式 | 费用（约） | 周期（约） |
|------|------------|------------|
| 自行申请 | 官费 ~¥300（视地区减免） | 2–3 个月 |
| 代理加急 | ¥800–¥2,000 | 2–4 周（加急另计） |

**用途**：华为、小米、OPPO、vivo 等商店上架常要求；应用商店版权申诉材料。

**材料**：源代码前后各 30 页、说明书、营业执照（个人开发者可用身份证 + 部分平台个人主体）。

### 6.2 ICP 备案

| 项 | 说明 |
|----|------|
| 费用 | 备案本身**免费**；需已备案域名 + 国内服务器（服务器 ¥50–200/月 起） |
| 周期 | 约 **7–20 个工作日**（各省管局不同） |
| 关联 | 若隐私政策托管在**国内域名**，通常需备案；纯 GitHub Pages 海外托管则 Play 海外不强制 |

### 6.3 国内安卓商店（若未来扩展）

| 项 | 说明 |
|----|------|
| 软著 | 多数主流商店**必需** |
| 隐私合规 | 隐私政策弹窗、权限说明、个人信息收集清单（你方 `assets/legal/` 可复用） |
| 额外 | 部分商店要求安全评估、版权承诺函；无统一国标，按渠道清单准备 |

**粗算**：个人开发者「软著加急 + 首年域名服务器 + Play 海外 \$25」约 **¥1,500–3,000 + \$25**；时间 **1–2 个月**可完成国内上架前置（不含开发）。

## 7. IAP 技术路线（v1.1 预备）

| 项 | 建议 |
|----|------|
| 插件 | `in_app_purchase`（官方）或 RevenueCat（简化校验） |
| SKU | 一个非消耗型：`pro_lifetime` |
| 内容包 | 消耗型或非消耗型：`dict_pack_full` 等 |
|  entitlement | 购买成功后写本地 + Play 购买 token 校验 |
| 恢复购买 | 设置页「恢复购买」入口（Play 要求） |
| 无账号 | 与 PRD「高隐私」一致；绑定 Google Play 账户即可 |

## 8. 指标与口碑（为何选此路线）

| 指标 | 预期 |
|------|------|
| 商店评分 | 无「看广告才能读」类差评 |
| 留存 | 阅读无硬墙，长篇完成率更真实 |
| 作品集叙事 | 「本地优先阅读器 + 可选 Pro 学习工具」 |
| 隐私 | 无广告 SDK 第三方追踪，Data safety 表单更简单 |
