### 4.2 新增替代章节（建议编号 §5.3）

**§5.3 商业模式（v0.3 新增）**

| 规则 | 说明 |
|------|------|
| v1.0 | 完全免费；**无广告**；**无阅读块数限制** |
| v1.1+ Pro | 一次性买断「Pro 永久」 |
| Pro 范围 | Anki 导出 → 匿名备份 → 大词典包（见 monetization §3.2） |
| 内容包 | 大词典等可选 IAP，与 Pro 可分售 |
| 不做 | 订阅强绑阅读、激励视频、章节付费墙 |

**§5.4 发布与平台（v0.3 新增）**

| 项 | 说明 |
|----|------|
| v1.0 首发 | Google Play 海外 |
| 国内渠道 | 非 v1.0 范围；软著/备案见 monetization §6 |
