# 帮助与反馈

## 常见问题

**如何导入书籍？**  
在书架点击「+」或前往「资源」Tab 查看导入教程，选择 EPUB/TXT 文件即可。

**如何设置词汇量？**  
首次启动会进入词库向导；之后可在「词库」Tab 或「个人 → 学习设置 → 管理词库」中调整。

**主页外观、阅读外观与阅读器浮层有何区别？**  
「主页外观」设置五个 Tab 内容区背景；「阅读外观」设置阅读器默认字号、行距与背景；阅读器内设置浮层可临时调节当前阅读，并与「阅读外观」共用同一套偏好。

**点击 ringing、was 等变形词为什么有时查不到？已会记哪个？**  
详见「个人 → 学习设置 → 词形查词说明」。查词会通过双 Tab 回落到原形；「已会」与高亮仅针对你选中的词形，不会随原形自动联动。
